Langkah pembuatan github page menggunakan sistem git

  1. Login ke akun Github.
  2. Membuat repository baru.
  3. Melakukan git bash pada file local.
  4. Membuka git bash pada repositori file local.
  5. Git init untuk menginisiasi repositori git kosong pada file local.
  6. Git status untuk memeriksa status pada folder tersebut.
  7. Git add untuk menambahkan semua file ke dalam staging area.
  8. Melakukan commit terhadap file yang sudah di add.
  9. Melakukan remote repositori GitHub dengan perintah git remote add origin.
  10. Melakukan push file ke dalam repository GitHub.
  11. Halaman web sudah di publish
