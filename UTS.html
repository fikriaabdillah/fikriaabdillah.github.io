Langkah pembuatan github page menggunakan sistem git

  1. Membuat folder local untuk dijadikan repositori.
  2. Melakukan git init untuk membuat repositori pada file local yang nantinya ada folder .git.
  3. Login pada akun github lalu membuat repositori baru.
  4. Membuka git bash pada repositori file local.
  5. Masuk ke folder di mana kita menyimpan file HTML, CSS, dan lain sebagainya.
  6. Memanggil perintah git add untuk menambahkan file ke staging area.
  7. Melakukan commit terhadap file yang sudah di-add.
  8. Melakukan push file ke repositori di GitHub.
  9. Setting bagian GithubPages dan kemudian link sudah bisa diakses
